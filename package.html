衣物（已抓洗衣輪替量）

冬季日間 15–19°C、清晨與沙漠夜晚可低到 7–8°C，採洋蔥式穿搭。以下為「每人」數量，已假設中途在 Bedford 洗一次衣，無須帶滿 11 天。

  • 睡衣 / 居家服2 套

洗衣相關

大本營有洗衣機，建議第 7–8 天（在伯斯期間）洗一次。投幣洗衣店在當地也很普遍。
  • 洗衣膠囊 / 旅行裝洗衣精數顆帶幾顆即可，當地超市也買得到
  • 曬衣繩 / 摺疊衣架1 組冬天不一定有烘乾，晾乾備用
  • 髒衣袋 / 大夾鏈袋2–3 個分裝髒衣、濕泳衣、沾沙衣物
  • 零錢（投幣洗衣 / 烘乾備用）澳幣硬幣，自助洗衣店常用

自駕用品

西澳路途長、鄉間訊號弱、黃昏有袋鼠出沒。安全與導航是重點；兒童安全座椅是法律規定。
★ 法規與安全（不可省）
  • 兒童安全座椅 / 增高墊★西澳法律：7 歲以下強制；建議租車時一併預訂，或自備合規座椅。確認數量符合兒童人數
  • 手機車架 + 車充★1–2 組導航與長途充電必備
  • 離線地圖下載（Google Maps）★卡巴里國家公園訊號極弱，行前先下載整區
  • 當地 SIM / eSIM / 漫遊每人或共用Telstra 收訊最廣；至少 1–2 人有網路
  • 行車記錄器（如租車未附）事故舉證用，視租車公司而定
  • 保溫水壺 / 大水瓶每人 1國家公園與 Rottnest 無補給，車上常備水
  • 車上零食 / 乾糧長途與沙漠路段補充能量
  • 墨鏡每人 1西曬刺眼、沙丘反光、行車安全
  • 垃圾袋 / 濕紙巾（車內）保持車內整潔，沾沙好清理
  • 記下租車公司 24h 道路救援電話拋錨、爆胎時聯絡

防曬・防風沙

澳洲紫外線冬天依然強；蘭斯林沙丘與尖峰石鎮風大時細沙漫天，需保護眼睛與 3C。
  • 高係數防曬乳 SPF50+★澳洲日照強，臉與手必擦
  • 帽子（遮陽 / 防風）每人 1
  • 護目鏡 / 魔術頭巾★滑沙與石鎮擋風沙；保護眼口鼻
  • 護唇膏 / 乳液數支冬季乾冷易乾裂
  • 相機 / 鏡頭防塵套沙丘換鏡頭格外小心，細沙無孔不入

藥品・健康

澳洲藥局買藥需英文且部分需處方，常用藥從台灣帶齊最省事。含兒童劑型要先備好。
  • 個人慢性病處方藥★足量+備份帶足全程並多備幾天；隨身行李攜帶
  • 感冒 / 退燒 / 止痛藥成人與兒童劑型分開準備
  • 暈車藥★長途自駕 + Rottnest 渡輪，兒童尤其需要
  • 腸胃藥 / 止瀉 / 整腸換食物水土不服備用
  • OK繃 / 簡易急救包 / 消毒健行擦傷、玩沙小傷口
  • 過敏藥 / 防蚊液戶外活動多
  • 藥品英文清單 / 成分說明入境申報與就醫溝通用

電子・3C

澳洲使用 Type I 三腳扁插、230V，與台灣不同，務必帶轉接頭。
  • 澳洲轉接頭（Type I）★2–3 個八字三腳扁插；多帶幾個分房充電
  • 延長線 / 多孔 USB 充電器1–2 個一個轉接頭即可同時充多裝置
  • 行動電源每 1–2 人 1 個須隨身（不可托運）
  • 各裝置充電線手機、相機、平板，逐一清點
  • 相機 / GoPro / 記憶卡 / 備用電池粉紅湖空拍、Quokka 自拍
  • 耳機 / 兒童平板（車上娛樂）長途安撫小孩

盥洗・日用

Airbnb/旅館備品不一，基本盥洗自備較安心；當地超市（Woolworths/Coles Inglewood）可隨時補。
  • 牙刷 / 牙膏每人
  • 洗髮 / 沐浴 / 護髮（旅行瓶）部分住宿不提供
  • 毛巾 / 快乾巾1–2 條露營地/部分住宿不附；浮潛後擦乾
  • 保養品 / 卸妝 / 化妝品乾冷氣候加強保濕
  • 衛生用品 / 濕紙巾 / 面紙
  • 指甲剪 / 棉花棒 / 小剪刀剪刀須托運
  • 兒童專用備品兒童牙膏、慣用安撫物、濕紙巾

錢財・雜項

澳洲多為刷卡 / 感應支付，少量現金備用即可。
  • 信用卡2 張以上分散風險；確認可海外感應支付
  • 澳幣現金少量投幣洗衣、小店、停車備用
  • 輕便後背包 / 隨身包每人下車看景點、環島、健行用
  • 折疊購物袋 / 保冷袋超市採買、自煮、買生鮮回程
  • 雨傘 / 輕便雨衣冬季偶有陣雨
  • 保溫瓶 / 隨行杯咖啡之國，環保杯方便
  • 兒童玩具 / 著色本 / 零食長途車程與餐廳等候安撫
  • 夾鏈袋 / 收納袋若干分裝、防水、裝沙土衣物

入境・檢疫提醒

澳洲檢疫（生物安全）極嚴，誤帶可重罰。打包與填申報卡前先確認。
  • 勿帶肉類 / 蛋 / 乳製品 / 生鮮蔬果 / 種子★含肉乾、肉鬆、新鮮水果，違者重罰
  • 如帶中藥 / 保健食品 / 泡麵主動申報，部分成分受限
  • 藥品隨身 + 成分說明大量或特殊藥物主動申報
  • 不確定就申報（走紅色通道）申報不罰、未申報才罰
